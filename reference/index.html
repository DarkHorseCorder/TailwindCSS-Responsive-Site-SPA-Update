UN NUEVO MODELO DE ATENCIÓN CIUDADANA

En esta página encontrarás el nuevo robot de Whatsapp "Leo" mediante el cuál podrás reportar, solicitar y dar seguimiento a los que ofrece la Presidencia Municipal. También podrás usar el buscador para encontrar otros canales de atención como nuestras apps, la página web de reporte ciudadano y nuestras redes sociales.
Qué es Hola León
Hola León es un nuevo modelo de atención ciudadana que integra instrumentos y acciones para servir con mayor cercanía, calidad y calidez a los leoneses.
